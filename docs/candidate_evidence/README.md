# Candidate Evidence Packages

This directory contains tracked candidate evidence packages that are useful for
paper planning but are not yet part of the current paper-freeze boundary.

Use this layer when a local evidence draft has enough structure to preserve, but
the source artifacts are still local generated outputs or the main claim has not
been promoted into `docs/PAPER_RESULT_PACKAGE.md`.

Promotion into the paper freeze requires all of the following in the same
change:

- source artifacts tracked or regenerated by a documented command;
- `.gitignore` unignore rules for any required generated outputs;
- `docs/PAPER_FREEZE_MANIFEST.md`, `docs/PAPER_RESULT_PACKAGE.md`,
  `docs/PROJECT_MAP.md` and `results/README.md` updates;
- `tests/mainline_artifact_checks.py` coverage for the new artifact chain;
- passing `make check` and `make mainline-audit`.
