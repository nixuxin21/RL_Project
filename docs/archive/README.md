# Archived Project Records

This directory keeps maintenance or audit records that are useful for traceability
but are no longer primary entry points for daily work, paper writing, or
reproduction.

Current files:

- `AUDIT_REPORT.md`: resolved project audit and historical risk register. The
  current maintenance source of truth is `docs/PROJECT_STATUS.md`; paper-facing
  result boundaries are in `docs/PAPER_FREEZE_MANIFEST.md`.
